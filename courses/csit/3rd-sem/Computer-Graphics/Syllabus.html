Course Discription & Objective
Course Discription:

The course coversconcepts of graphics hardware, software, and applications,data structures for representing 2D and 3D geometric objects, drawing algorithms for graphical objects, techniques for representing and manipulating geometric objects, illumination and lighting models, and concept of virtual reality.


Course Objectives:

The objective of this course is to understand the theoretical foundation as wellas the practical applications of 2D and 3D graphics. Units and Unit Content

Units and Unit Contents
Introduction of Computer Graphics
3 hrs
  1. 1.1 A Brief Overview of Computer Graphics, Areas of Applications.
  2. 1.2 Graphics Hardware: Display Technology, Architecture of Raster-Scan Displays,Vector Displays, Display Processors, Hard copy device. Input Devices.
  3. 1.3 Graphics Software: Software standards, Need of machine independent graphics language.
Scan Conversion Algorithm
6 hrs
  1. 2.1 Scan Converting a Point and a straight Line: DDA Line Algorithm, Bresenham’s Line Algorithm
  2. 2.2 Scan Converting Circle and Ellipse :Mid Point Circle and Ellipse Algorithm
  3. 2.3 Area Filling: Scan Line Polygon fill Algorithm, Inside-outside Test, Scan line fill of Curved Boundary area, Boundary-fill and Flood-fill algorithm
Two-Dimensional Geometric Transformations
5 hrs
  1. 3.1 Two-Dimensional translation, Rotation, Scaling, Reflection and Shearing
  2. 3.2 Homogeneous Coordinate and 2D Composite Transformations. Transformation between Co-ordinate Systems.
  3. 3.3 Two Dimensional Viewing: Viewing pipeline, Window to viewport coordinate transformation
  4. 3.4 Clipping: Point, Lines(Cohen Sutherland line clipping, Liang-Barsky Line Clipping) , Polygon Clipping(Sutherland Hodgeman polygon clipping)
Three-Dimensional Geometric Transformation
5 hrs
  1. 4.1 Three-Dimensional translation, Rotation, Scaling, Reflection and Shearing
  2. 4.2 Three-Dimensional Composite Transformations
  3. 4.3 Three-Dimensional Viewing: Viewing pipeline, world to screen viewing transformation, Projection concepts(Orthographic, parallel, perspective projections)
3D Objects Representation
7 hrs
  1. 5.1 Representing Surfaces: Boundary and Space partitioning
  2. 5.1.1 Polygon Surface: Polygon tables , Surface normal and Spatial orientation of surfaces, Plane equations, Polygon meshes
  3. 5.1.2 Wireframe Representation
  4. 5.1.3 Blobby Objects
  5. 5.2 Representing Curves: Parametric Cubic Curves, Spline Representation, Cubic spline interpolation, Hermite Curves, Bezier and B-spline Curve and surface
  6. 5.3 Quadric Surface: Sphere and Ellipsoid
Solid Modeling
4 hrs
  1. 6.1 Sweep ,Boundary and Spatial-Partitioning Representation
  2. 6.2 Binary Space Partition Trees (BSP)
  3. 6.3 Octree Representation
Visible Surface Detections
5 hrs
  1. 7.1 Image Space and Object Space Techniques
  2. 7.2 Back Face Detection, Depth Buffer (Z-buffer), A-Buffer and Scan-Line Algorithms.
  3. 7.3 Depth Sorting Method (Painter’s Algorithm)
  4. 7.4 BSP tree Method, Octree and Ray Tracing
Illumination Models and Surface Rendering Techniq
5 hrs
  1. 8.1 Basic Illumination Models: Ambient light, Diffuse reflection, Specular reflection and Phong model
  2. 8.2 Intensity attenuation and Color consideration ,Transparency, Shadows
  3. 8.3 Polygon Rendering Methods : Constant intensity shading, Gouraud shading , Phong Shading and Fast Phong Shading
Introduction to Virtual Reality
2 hrs
  1. 9.1 Concept of Virtual reality
  2. 9.2 Virtual Reality Components of VR System, Types of VR System, 3D Position Trackers, Navigation and Manipulation Interfaces
  3. 9.3 Application of VR
Introduction to OpenGL
3 hrs
  1. 10.1 Introduction, Callback functions, Color commands, Drawings pixels, lines, polygons using OpenGL, Viewing and Lighting
Lab and Practical Works
Laboratory Works:

Students should be able to write program on the most of the contents listed in syllabus, using any known programming language (C, C++) in previous semester. Majorly, students should on computer graphics primitives like line, circle and ellipse drawing algorithm to hidden surface removal techniques. After completing the basic lab session the students must be able to design some project works like game, 3D rotation, screen saver etc. Some sample lab sessions can be as following:-

    Unit 2 :
  1. Study of Fundamental Graphics Functions
  2. Implementation of Line drawing algorithms: DDA Algorithm, Bresenham's Algorithm
  3. Implementation of Circle drawing algorithms: Bresenham's Algorithm, Mid-Point Algorithm
    Unit 3 :
  1. Simulation of 2D transformation, Rotation and Scaling
  2. Write a program to implement Cohen Sutherland line clipping algorithm
    Unit 4 :
  1. Write a program to perform shear transformation on a rectangle
  2. Write a program to perform 2D Transformation on a line
  3. Write a program to draw a car using in build graphics function and translate it from bottom left corner to right bottom corner of screen
  4. Write a program to draw a cube using in build library function and perform 3D transformations
    1. Translations in x, y, z directions
    2. Rotation by angle 450 about z axis, rotation by 600 about y-axis in succession.
    3. Scaling in x-direction by a factor of 2, scaling in y- direction by a factor of 3
    Unit 5 :
  1. Implementation of polygon tables.
  2. Write a program to draw Bezier curve, sphere
    Unit 7 :
  1. Back face detection:- Implementation of Depth Buffer, A – Buffer, Scan-Line algorithm
  2. Implementation of rotation of 3D cube.
    Unit 10 :
  1. Event driven programming
  2. Point, Line and Polygon
  3. Drawing 3D objects
